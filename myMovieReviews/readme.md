## 추가로 구현한 기능:
- 리뷰 리스트 페이지에서 제목, 별점, 러닝타임 등 정렬 기능 추가.
- 장르를 단순 입력이 아닌 선택 기능으로 변경하기!
- 리스트 페이지 정렬 기능 넣기!(제목 이름 순, 별점 순, 상영 시간 순 등)

- 부트스트랩 이용하여 프론트 디자인 개선
- 커스텀 404 페이지 추가
